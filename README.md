🧭 The Index - Prescript Generator

"Execute the Will of the Prescript without question."

A lore-accurate, interactive web application inspired by *Library of Ruina*. This application interfaces with **The Index** to deliver divine instructions directly to the Proxies. Featuring visual distortions, scrambled textual decoding, auditory indicators, and anti-tamper security layers to protect the transmission.

== 🎭 Features ==

* 📜 Authentic Prescripts: Contains a vast collection of dictates, spanning from the mundane to the ominous and highly specific.
* ⚡ Glitch Audio Syncing: Synthesized audio cues match textual reveal states, modulating visually on decrypt wavefronts.
* 🛡️ Proxies Protection Layer: Built-in security layers to detect and block modern Developer Tools to prevent the distortion of transmissions.
* 📱 Responsive Interface: Designed to accommodate both terminal displays and handheld mobile devices.

== 🛠️ Built With ==

* HTML5 - Structured semantic elements (`aria-live` accessible text wrappers).
* CSS3 - Monochromatic custom aesthetic with glowing CSS drop-shadow filters.
* Vanilla JavaScript - High-performance animations powered by `requestAnimationFrame`.
